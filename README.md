# Moringa Toolkit - TailwindCSS Beginner's Guide

## 🎓 Moringa AI Capstone Project

A comprehensive beginner's guide to TailwindCSS in React, built with Vite and featuring a modern, responsive design.

## 🚀 Quick Start

### Prerequisites
- Node.js v18 or higher
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will be available at `http://localhost:8080`
