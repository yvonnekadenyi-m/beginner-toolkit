# Beginner Toolkit: AI-Themed Graduate Welcome in Go

## Objective
Create a beginner-friendly Go command-line program that welcomes Moringa AI graduates and displays a randomly selected AI-themed motivational message.

- **Why Go?** Simple syntax, beginner-friendly, strong performance.
- **Why this project?** It’s a tiny but unique program that blends AI concepts with graduate encouragement.
- **End goal:** Print a dynamic, inspiring message for graduates each time the program runs.
